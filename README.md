## 💻 Sobre o Projeto

Este projeto consiste em uma landing page de alto padrão para um e-commerce de calçados finos. O objetivo central foi criar uma experiência de compra fluida e sofisticada, unindo um design refinado no Figma com uma implementação técnica sólida e otimizada, seguindo rigorosamente a estratégia **Mobile-First**.

---

## 🛠️ Tecnologias e Ferramentas

Para a construção deste projeto, utilizei:

* <img src="https://skillicons.dev/icons?i=html" height="32"> **HTML5:** Estrutura semântica e foco em SEO.
* <img src="https://skillicons.dev/icons?i=css" height="32"> **CSS3:** Estilização avançada com Flexbox e Grid.
* <img src="https://skillicons.dev/icons?i=js" height="32"> **JavaScript:** Lógica para interatividade e controle do menu.
* <img src="https://skillicons.dev/icons?i=figma" height="32"> **Figma:** Prototipagem de alta fidelidade e Design System.
* <img src="https://skillicons.dev/icons?i=font-awesome" height="32"> **Font Awesome:** Ícones para uma interface intuitiva.
