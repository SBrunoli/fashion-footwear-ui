# # Bella Passos — E-commerce Front-end

Este projeto consiste em uma landing page de alto padrão para um e-commerce de calçados finos. O objetivo central foi criar uma experiência de compra fluida e sofisticada, unindo um design refinado no Figma com uma implementação técnica sólida e otimizada, seguindo rigorosamente a estratégia **Mobile-First**.

---

## 🚀 Do Protótipo à Realidade

O fluxo de trabalho englobou todo o ciclo de criação de um produto digital, focando na fidelidade entre o design planejado e o código final:

-   **🎨 UI/UX Design**: Prototipagem de alta fidelidade realizada no Figma, definindo o Design System.
-   **💻 Desenvolvimento**: Tradução técnica precisa para HTML5 e CSS3, com foco em semântica e performance.
-   **🧠 Interatividade**: Implementação de lógica de navegação e controle de estado do menu lateral utilizando JavaScript puro (Vanilla JS).

## 🌟 Funcionalidades em Destaque

| Categoria | Descrição da Funcionalidade |
| :--- | :--- |
| **Menu Mobile** | Sistema de menu lateral (Hambúrguer) interativo com controle de scroll para UX aprimorada em dispositivos móveis. |
| **Grid Responsivo** | Vitrine de produtos com grade dinâmica que se adapta automaticamente a qualquer tamanho de tela (Grid/Flexbox). |
| **Identidade Visual** | Implementação de Design System com tipografia refinada e Favicon para acabamento profissional. |
| **Performance** | Organização rigorosa de assets (SVGs e imagens) e código CSS modularizado para carregamento rápido. |
